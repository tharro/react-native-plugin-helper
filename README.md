# react-native-plugin-helper

Plugin for React native

## Installation

```sh
npm install react-native-plugin-helper
```

## Usage

```js
import { isValidateEmail } from 'react-native-plugin-helper';

// ...

const isValid = isValidateEmail('example@email.com');
```
